# 虚拟细胞论文建模提取 - 配置指南

## 前置条件

- Claude Code/Codex 可读取论文 PDF 或全文。
- 推荐安装 `pdftotext` 或同类 PDF 文本抽取工具。
- 若要自动验证 HTML，推荐安装 Node.js 与 Playwright。

## 放置方式

将整个目录复制到技能目录：

```bash
cp -r skills/虚拟细胞论文建模提取 ~/.claude/commands/
```

## 使用示例

```text
用「虚拟细胞论文建模提取」读这篇 E. coli 代谢工程论文，输出交互式 markmap HTML。
```

```text
把这篇嘌呤代谢论文提取成虚拟细胞建模信息，重点看 iML1515 里能怎么表示。
```

## 输出检查

- HTML 能打开。
- markmap 工具栏可见。
- 至少包含研究对象、通路、基因/蛋白、代谢物、实验数据、建模假设、边界缺口。
- 至少 8 个原文事实经过核对。

## 常见问题

- 如果论文是综述：输出时必须区分“综述背景”和“可直接用于 E. coli 模型的证据”。
- 如果找不到 reaction ID：先列候选名称和需要查询的数据库，不要编造 ID。
- 如果只有表达量数据：标注为近似建模，不能直接当作 flux。
