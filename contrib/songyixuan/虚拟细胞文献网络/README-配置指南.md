# 虚拟细胞文献网络 - 配置指南

## 前置条件

- 至少两篇论文或已有速读/脑图结果。
- 推荐先用「虚拟细胞论文建模提取」处理单篇论文，再用本技能联通多篇论文。

## 放置方式

```bash
cp -r skills/虚拟细胞文献网络 ~/.claude/commands/
```

## 使用示例

```text
用「虚拟细胞文献网络」把 Zhang 2024、Grove 2025、Gedeon 2023 三篇论文联通起来，输出交互式 markmap HTML。
```

```text
比较这几篇 E. coli 嘌呤代谢论文，找共同基因、代谢物、通路和可合并的建模假设。
```

## 输出检查

- HTML 能打开，markmap 工具栏可用。
- 每篇论文都有明确“角色”。
- 共同实体按基因/代谢物/通路/表型分类。
- 每条跨论文连接都标注支持、扩展、限制、冲突或缺口。

## 常见问题

- 不要把不同物种的调控机制直接合并。
- 不要把综述证据和原始实验数据混为同一证据强度。
- 如果连接不充分，应把它列为缺口，而不是强行生成结论。
